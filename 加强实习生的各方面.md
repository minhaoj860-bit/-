## 五、排障思路训练

### 排障六步法（必须遵循）

```
1. 清晰描述现象（是什么？）
2. 确认期望行为（应该是什么？）
3. 分析差异点（哪里不一样？）
4. 提出假设（可能是什么原因？）
5. 验证假设（用什么命令验证？）
6. 定位根因（最终原因是什么？）
```

### 常用排障命令

| 用途 | 命令 |
|------|------|
| 端口连通性 | nc -zv host port |
| 网络抓包 | tcpdump -i interface |
| 进程查看 | ss -tlnp |
| 日志查看 | journalctl -u service |
| Docker状态 | docker info |
| Swarm节点 | docker node ls |

## 六、常用命令速查

### 网络排查

```bash
# 测试端口连通性
nc -zv 10.0.0.1 2377

# 查看端口监听
ss -tlnp | grep 2375

# 测试UDP连通性
nc -vzu 10.0.0.1 51820

# 查看网络接口
ip addr show
```

### Docker排查

```bash
# 查看Docker状态
docker info | grep -i swarm

# 查看节点
docker node ls

# 查看服务
docker service ls

# 查看容器日志
docker logs <container-name>
```

### 系统排查

```bash
# 查看服务状态
systemctl status docker

# 查看进程
ps aux | grep docker

# 查看防火墙
iptables -L -n
```
